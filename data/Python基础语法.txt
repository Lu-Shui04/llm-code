# Python 基础语法教程

## 1. 变量与数据类型
Python 中变量不需要声明类型，直接赋值即可使用。
常见的数据类型有：整数（int）、浮点数（float）、字符串（str）、布尔值（bool）。
例如：name = "张三"、age = 20、height = 1.75、is_student = True。
使用 type() 函数可以查看变量的类型。

## 2. 条件判断
条件判断使用 if、elif、else 关键字。
语法格式为：if 条件: 执行语句，注意冒号和缩进。
比较运算符包括：==, !=, >, <, >=, <=。
逻辑运算符 and、or、not 可以组合多个条件。

## 3. 循环结构
Python 有两种循环方式：for 循环和 while 循环。
for 循环用于遍历序列（列表、元组、字符串等）。
while 循环在条件为真时一直执行，需要注意避免死循环。
使用 break 可以提前退出循环，continue 跳过当前次循环。

## 4. 列表与元组
列表用方括号 [] 表示，元素可以修改。
元组用圆括号 () 表示，元素不可修改。
列表常用方法：append() 追加、insert() 插入、pop() 弹出、remove() 删除。
使用索引和切片可以访问序列中的元素。

## 5. 字典与集合
字典是键值对结构，用花括号 {} 表示。
例如：student = {"name": "小明", "age": 18}。
通过 student["name"] 可以获取字典中的值。
集合是无序不重复元素的集合，支持交集、并集等操作。

## 6. 函数定义
使用 def 关键字定义函数。
函数可以有参数和返回值。
使用 return 语句可以返回计算结果。
Python 还支持匿名函数 lambda，用于简单的函数定义。

## 7. 文件操作
使用 open() 函数打开文件，参数指定文件路径和模式。
读取模式 "r"、写入模式 "w"、追加模式 "a"。
使用 with 语句可以自动关闭文件，推荐使用。
read() 读取全部内容，readlines() 按行读取返回列表。
