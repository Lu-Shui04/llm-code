Linux 基本命令
================

一、目录操作
-----------
ls          列出当前目录下的文件和文件夹
ls -l       详细列表（显示权限、大小等）
ls -a       显示隐藏文件（.开头的文件）
pwd         显示当前所在目录的完整路径
cd 目录名   进入某个目录
cd ..       返回上一级目录
cd ~        回到用户主目录
mkdir 名    创建新目录

二、文件操作
-----------
touch a.txt      创建空文件
cat a.txt        查看文件全部内容
head a.txt       查看文件前10行（head -5 a.txt 前5行）
tail a.txt       查看文件后10行
rm a.txt         删除文件
rm -r 目录名     删除目录及其内容
cp a.txt b.txt   复制文件 a → b
mv a.txt b.txt   移动/重命名文件 a → b

三、文件编辑（vim）
-----------------
vim a.txt        打开文件
按 i             进入编辑模式
按 Esc           退出编辑模式
输入 :wq         保存并退出
输入 :q!         不保存强制退出

四、权限
-----------
chmod 755 a.txt  设置文件权限（可读可执行）
chmod +x a.py    给文件加执行权限
ls -l            第一列就是权限（如 -rwxr-xr-x）

五、查找
-----------
find . -name "*.py"   在当前目录下找所有 .py 文件
grep "关键词" a.txt   在文件中搜索关键词
grep -r "关键词" .    在当前目录所有文件里搜

六、管道和重定向
---------------
echo "hello" > a.txt   把输出写入文件（覆盖）
echo "hello" >> a.txt  把输出追加到文件末尾

七、下载和网络
--------------
curl 网址              请求一个网址（显示内容）
wget 网址              下载文件
ping 域名              测试网络连通性
