## 📋 学习路线

| 🎯 阶段 | 📖 内容 |
|----------|---------|
| Python 基础 | 变量、条件判断、循环、列表字典、函数、文件操作、异常处理 |
| 模型调用 | OpenAI SDK、LangChain、Ollama 本地部署大模型、流式输出 |
| 前端界面 | Streamlit、HTML/CSS 基础 |
| RAG 技术栈 | 文档解析、文本切块、向量嵌入、向量数据库、知识库问答系统 |
| AI Agent | Function Calling、工具调用、多智能体协作 |
| 后端 & 部署 | FastAPI、Docker 基础、Linux |
| 项目 & 面试 | 2-3 个完整项目、简历打磨、技术面试 |

> 简单的快速过，难的慢慢啃。不卡进度，卡质量。
